## Module Data.Iso

#### `Iso`

``` purescript
data Iso a b
  = Iso (a -> b) (b -> a)
```

An isomorphism between types `a` and `b` consists of a pair of functions
`f :: a -> b` and `g :: b -> a`, satisfying the following laws:

- `f <<< g = identity`
- `g <<< f = identity`

The isomorphisms in this library satisfy these laws by construction.
